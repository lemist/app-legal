Suporte do NoteWhen

Esta página explica como obter ajuda com o NoteWhen e responde dúvidas comuns sobre o uso do app.

Fale conosco

Se você precisa de ajuda, quer reportar um bug ou solicitar um recurso, entre em contato por GitHub Issues.

Tópicos comuns

Primeiros passos

O NoteWhen permite definir eventos de saúde (como sintomas, medicamentos, sono ou exercícios) e registrar cada ocorrência com data/hora e notas opcionais. Use o botão + na aba Eventos para adicionar eventos, toque em um evento para ver o histórico ou use o widget para registro rápido.

Sincronização com iCloud

Seus dados ficam armazenados no seu dispositivo e são sincronizados pela sua conta privada do iCloud quando você está conectado ao iCloud. Se os dados não aparecerem em outro dispositivo, confirme se você está usando o mesmo Apple ID e se o iCloud está ativado para o NoteWhen nos Ajustes do iOS.

Apple Health

Você pode importar treinos, sono e água do Apple Health e, opcionalmente, salvar no Apple Health os registros feitos no NoteWhen. Gerencie permissões de leitura e gravação no app Saúde ou nos Ajustes do iOS. Se o acesso foi negado, abra a tela de detalhes de um evento e use os controles de sincronização para solicitar acesso novamente.

Lembretes

Configure lembretes de registro e de medicação em Ajustes. Os lembretes são enviados apenas como notificações locais no seu dispositivo.

Widget

Adicione o widget do NoteWhen para registrar eventos rapidamente pela Tela de Início. O widget mostra atalhos para seus eventos ativos. O histórico de eventos é armazenado no app principal, não no widget.

Relatórios mensais com IA

Relatórios mensais opcionais resumem seus eventos acompanhados usando IA. Os relatórios exigem uma assinatura ativa. Os relatórios gerados são salvos no app e sincronizados via iCloud. A saída da IA é apenas para referência pessoal, não constitui orientação médica.

Assinaturas

Gerencie as assinaturas de relatórios mensais com IA em Ajustes > Apple ID > Assinaturas. As compras são processadas pela Apple; não armazenamos os dados do seu cartão de pagamento.

Privacidade

Consulte nossa Política de Privacidade para saber como o NoteWhen trata suas informações.

Aviso médico

O NoteWhen é apenas para acompanhamento pessoal. Não é um dispositivo médico. Consulte um profissional de saúde qualificado para decisões médicas.
