Assistance NoteWhen

Cette page explique comment obtenir de l'aide pour NoteWhen et répond aux questions fréquentes sur l'utilisation de l'application.

Nous contacter

Si vous avez besoin d'aide, souhaitez signaler un bug ou demander une fonctionnalité, contactez-nous via GitHub Issues.

Sujets fréquents

Premiers pas

NoteWhen vous permet de définir des événements de santé (comme des symptômes, des médicaments, le sommeil ou l'exercice) et d'enregistrer chaque occurrence avec un horodatage et des notes facultatives. Utilisez le bouton + dans l'onglet Événements pour ajouter des événements, touchez un événement pour voir l'historique, ou utilisez le widget pour un enregistrement rapide.

Synchronisation iCloud

Vos données sont stockées sur votre appareil et synchronisées via votre compte iCloud privé lorsque vous êtes connecté à iCloud. Si des données n'apparaissent pas sur un autre appareil, vérifiez que vous utilisez le même identifiant Apple et qu'iCloud est activé pour NoteWhen dans Réglages iOS.

Apple Health

Vous pouvez importer les entraînements, le sommeil et l'eau depuis Apple Health, et enregistrer en option dans Apple Health les entrées saisies dans NoteWhen. Gérez les autorisations de lecture et d'écriture dans l'app Santé ou dans Réglages iOS. Si l'accès a été refusé, ouvrez l'écran de détail d'un événement et utilisez les contrôles de synchronisation pour redemander l'accès.

Rappels

Configurez les rappels de suivi et de médicaments dans Réglages. Les rappels sont envoyés uniquement sous forme de notifications locales sur votre appareil.

Widget

Ajoutez le widget NoteWhen pour enregistrer rapidement des événements depuis votre écran d'accueil. Le widget affiche des raccourcis pour vos événements actifs. L'historique des événements est stocké dans l'application principale, pas dans le widget.

Rapports mensuels IA

Les rapports mensuels facultatifs résument vos événements suivis à l'aide de l'IA. Les rapports nécessitent un abonnement actif. Les rapports générés sont enregistrés dans l'application et synchronisés via iCloud. Les résultats de l'IA sont uniquement fournis à titre de référence personnelle et ne constituent pas un avis médical.

Abonnements

Gérez les abonnements aux rapports mensuels IA dans Réglages > Identifiant Apple > Abonnements. Les achats sont traités par Apple ; nous ne stockons pas les informations de votre carte de paiement.

Confidentialité

Consultez notre Politique de confidentialité pour savoir comment NoteWhen traite vos informations.

Note médicale

NoteWhen est destiné au suivi personnel uniquement. Ce n'est pas un dispositif médical. Consultez un professionnel de santé qualifié pour toute décision médicale.
